We can provide styling by defining the tag. What this does is it provides the same style for the given tags in the document

The other way to style an element is using the id.It is an attribute for the tag. Each tag can only have one id.

Styling can also be provided using the classes and this is set using the class attribute of the tag and there could be multiple tag with same class and the styling would be applied to all the tags with the given class.
